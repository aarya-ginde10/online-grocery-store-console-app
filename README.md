## Project Title
Develop a system for a grocery store where users can purchase groceries and 
household items from the comfort of their homes.

## Project Description
This project is a Python console-based Online Grocery Store System that allows users to:
- Register and login
- Search groceries by name or category
- Add items to a shopping cart
- Checkout and place orders
- View order history and track order status

The system is implemented using Object-Oriented Programming (OOPs), 
CRUD operations, data structures, and file handling.

---

## Features
- User Registration & Login
- Product Search (Name & Category)
- Shopping Cart Management
- Checkout & Order Placement
- Order History Tracking
- Persistent Data Storage using JSON files

---

## Technologies Used
- Python 3.x
- Object-Oriented Programming (OOPs)
- JSON File Handling
- Lists, Dictionaries
- Console-based Menu System
